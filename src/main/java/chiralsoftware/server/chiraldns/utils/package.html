Classes which implement important DNS services, to be executed as stand-alone programs by users or used as threads within other software.

chiralDNS implements a stand-alone authoritative DNS server. It is a daemon which can be initiated from the command line, or could be used as a thread in another server.

Dig implements a utility similar to the BIND dig utility, meant to be accessed from the command line. Dig gives a detailed dump of the DNS packets it receives, allowing inspection of flags and compression.

LoadFile takes a data file, in a particular format, and stores it in the database. LoadFile is meant to be used from the command line to load zone data into the database.
